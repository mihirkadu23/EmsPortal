# EmsPortal
Developed a web-based Employee Management System (EMS Portal) using ASP.NET Core MVC, C#, SQL Server, Entity Framework Core, HTML, CSS, Bootstrap, and JavaScript. The application provides a centralized platform for managing employee records efficiently. Implemented features such as employee registration, viewing employee details, updating and deleting records (CRUD operations), search functionality, and secure database integration using Entity Framework Core. Designed a responsive and user-friendly interface while following the MVC architecture and clean coding practices.
